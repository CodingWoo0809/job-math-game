# 3map2 — 사방 투상 보안실

> 챕터: 3 — 도형과 측정  
> 주 성취기준: [12직수03-02]  
> 난이도: 3/5, 3단계 주관식 판독형  
> 현재 상태: 플레이 가능 웹 프로토타입 구현

## 1. 한 문장 정의

WH는 정육면체 적재 리프트의 평면도·정면도·우측면도·좌측면도를 함께 읽고, 구조물의 총 층수와 2층에 놓인 정육면체 수를 직접 입력하여 세 번의 하중 인증을 통과한다.

## 2. 이야기 상황

- 3map1의 문을 통과한 WH가 사방에 투상 카메라가 설치된 A-02 격실에 들어간다.
- 바닥에는 정육면체로만 물건을 적재하는 화물 리프트가 있다.
- 입체 스캐너가 고장 나 실제 구조물 영상은 사라지고 네 방향 투상도만 남아 있다.
- 리프트를 작동하려면 구조물의 총 층수와 2층 정육면체 수를 입력해 하중을 확인해야 한다.
- 기초·응용·정밀 판독을 한 문제씩 해결하면 리프트가 상승하여 3map3으로 이어진다.

## 3. 투상도 약속

- 평면도: 입체도형을 위에서 바라본 모양이다. 화면 아래쪽을 구조물의 앞쪽으로 둔다.
- 정면도: 입체도형을 앞에서 바라본 모양이다.
- 우측면도: 정면을 기준으로 구조물의 오른쪽에서 바라본 모양이다.
- 좌측면도: 정면을 기준으로 구조물의 왼쪽에서 바라본 모양이다.
- 측면도의 앞·뒤 방향은 도면 아래에 함께 표시한다.

## 4. 문제 규칙

- 모든 구조물은 같은 크기의 정육면체를 바닥부터 빈틈없이 쌓아 만든다.
- 한 문제에 네 투상도를 모두 제공한다.
- 플레이어는 `총 층수`와 `2층에 있는 정육면체 수`를 보기 없이 직접 입력한다.
- 총 층수는 정면도와 측면도에서 보이는 가장 높은 열로 판단한다.
- 2층 정육면체 수는 높이가 2 이상인 정육면체 기둥의 수와 같다.
- 각 문제는 네 투상도만으로 2층 정육면체 수가 하나로 결정되는 구조만 사용한다.
- 오답이면 현재 문제를 다시 풀며, 틀린 값에 맞춘 힌트를 제공한다.
- 기초 1문제, 응용 1문제, 정밀 1문제를 모두 해결하면 맵을 완료한다.

## 5. 문제 은행

| 난이도 | 문항 수 | 평면 격자 범위 | 학습 초점 |
|---|---:|---:|---|
| 기초 | 3 | 3×3 | 기준 방향과 가장 높은 층 찾기 |
| 응용 | 3 | 4×4~5×5 | 떨어진 기둥의 높이 대응하기 |
| 정밀 | 3 | 5×5~7×7 | 네 투상도를 동시에 비교하기 |
| 합계 | 9 | - | 매 플레이 3문항 무작위 출제 |

## 6. 완료 조건

- 9개 구조물의 평면도·정면도·우측면도·좌측면도가 원본 높이 자료와 일치한다.
- 모든 문제에서 투상도로 가능한 2층 정육면체 수가 정확히 하나다.
- 두 주관식 답을 모두 맞혀야 다음 문제로 이동한다.
- 세 난이도가 기초 → 응용 → 정밀 순서로 출제된다.
- 3문제를 해결한 뒤에만 3map3 이동 버튼이 나타난다.
