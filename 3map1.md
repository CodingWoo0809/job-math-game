# 3map1 — 접히는 방의 첫 번째 문

> 챕터: 3 — 도형과 측정  
> 주 성취기준: [12직수03-01]  
> 난이도: 3/5, 연속 판별형  
> 현재 상태: 플레이 가능 웹 프로토타입 구현

## 1. 한 문장 정의

WH는 다면체 격실의 벽에 투사되는 전개도를 보고, 이를 접었을 때 만들어지는 입체도형의 겨냥도를 다섯 보기에서 골라 5문제를 연속으로 맞혀 첫 번째 문을 연다.

## 2. 문제 규칙

- 문제 은행에는 서로 다른 전개도 50개가 있다.
- 매 문제는 전개도 1개와 겨냥도 보기 5개를 제시한다.
- 입체도형 이름은 보기에서 글자로 제공하지 않는다.
- 각 시도는 `쉬운 문제 2개 → 생각 문제 2개 → 고난도 문제 1개` 순서로 출제된다.
- 같은 난이도 안에서 문제와 보기 순서는 무작위다.
- 난이도별 문제 덱을 모두 사용하기 전에는 같은 문항 ID가 다시 나오지 않는다.
- 50문항을 모두 사용하면 덱을 다시 섞되, 직전 세트의 문항은 바로 반복하지 않는다.
- 겨냥도는 보이는 모서리를 실선, 보이지 않는 모서리를 점선으로 나타낸다.
- 전개도의 모든 면은 적어도 한 변을 다른 면과 공유하며 하나로 연결된다.
- 정답을 맞히면 연속 인증이 1 증가한다.
- 오답을 고르면 연속 인증이 0으로 초기화되고 다시 1번 문제부터 시작한다.
- 5문제를 연속으로 맞히면 격실 문이 열린다.

## 3. 문제 은행 구성

| 입체도형 | 전개도 수 |
|---|---:|
| 쉬움 — 정육면체 | 8 |
| 쉬움 — 직육면체 | 7 |
| 쉬움 — 원뿔 | 5 |
| 생각 — 삼각기둥 | 7 |
| 생각 — 사각뿔대 | 7 |
| 생각 — 오면체(사각뿔) | 6 |
| 고난도 — 정십이면체 | 5 |
| 고난도 — 정이십면체 | 5 |
| 합계 | 50 |

## 4. 플레이 흐름

1. WH가 다면체 격실 A-01에서 깨어난다.
2. 벽면 스캐너가 연결된 첫 번째 전개도와 실선·점선 겨냥도 다섯 개를 투사한다.
3. 플레이어가 면의 모양과 개수를 비교해 알맞은 겨냥도를 선택한다.
4. 정답이면 인증 표시등이 하나 켜지고 다음 무작위 문제가 나온다.
5. 1~2번은 쉬운 입체도형, 3~4번은 생각이 필요한 입체도형, 5번은 정십이면체나 정이십면체가 나온다.
6. 오답이면 표시등이 모두 꺼지고 난이도 순서를 유지한 새로운 1번 문제가 나온다.
7. 연속 5회 정답을 달성하면 문이 열리고 3map2 격실로 이동한다.

## 5. 완료 조건

- 50개의 문제 데이터가 중복 없는 ID로 구성된다.
- 쉬움 20개, 생각 20개, 고난도 10개로 문제 은행을 구분한다.
- 10개의 연속 세트를 생성하면 50문항이 중복 없이 모두 한 번씩 사용된다.
- 각 문제의 선택지는 서로 다른 겨냥도 5개이며 정답은 정확히 하나다.
- 모든 겨냥도에 실선과 점선이 모두 포함된다.
- 오답 시 진행도가 즉시 0/5로 초기화된다.
- 연속 5회 정답일 때만 완료 화면이 나타난다.
- 키보드와 포인터로 선택할 수 있으며 선택 결과를 색상 외 문장으로도 알린다.
